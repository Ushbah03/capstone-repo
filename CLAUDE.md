# Capstone Project Rules

## Tech Stack
- Frontend: React / Next.js
- Runtime: Node.js

## Coding Guidelines
- Use clean, modular components.
- Follow semantic HTML practices.
- Maintain clear, concise documentation.

### Project Rules Learned from Workflow Drill
- Always enforce epsilon buffers (`EPSILON = 0.00001`) when parsing floating-point percentage math operations.
- All interface input layouts must explicitly implement descriptive semantic ARIA properties (`aria-labelledby`, roles).
- Form states must implement immediate boundary restrictions (e.g., rejecting out-of-bounds inputs) before mutating arrays.
